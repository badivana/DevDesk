## 📖 About

DevDesk is a modern full-stack developer dashboard designed to provide developers with a single workspace for coding analytics, productivity, and system monitoring.

It integrates GitHub analytics, coding platform statistics, productivity tools, AI-powered utilities, and system information into one elegant dashboard.

The application is also designed to run as a **Wallpaper Engine** wallpaper, allowing developers to monitor their progress directly from their desktop.
